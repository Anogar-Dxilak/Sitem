## ⚡ Öne Çıkan Özellikler

* 💻 **İnteraktif Kali Linux Terminali (`guest@egemen`):**
  - Web tarayıcısı üzerinden çalışan sanal dosya sistemi.
  - `help`, `ls`, `cat`, `clear`, `whoami` gibi temel komutları ve dosya navigasyonunu destekler.
* 🌍 **Canlı Siber Saldırı Haritası & Matrix Arka Planı:**
  - Dinamik harita ve siber saldırı simülasyon animasyonu.
* 💬 **Firebase Realtime Database & E-posta Bildirimli Geri Bildirim:**
  - Ziyaretçilerin isim, kategori ve 5 yıldızlı değerlendirme ile yorum bırakabileceği form.
  - Gönderilen her geri bildirim için Firebase Realtime Database kaydı ve e-posta bildirimi.
* 📜 **Sertifika & Deneyim Listesi:**
  - Cisco CyberOps, Certified Cybersecurity Foundations, ISO 27001 sertifikaları ve SOC/Staj deneyimleri.
* 🎨 **Koyu Siber Tema (Dark Cyber Aesthetics):**
  - `#00ff66` neon yeşil vurgular, cam efektleri (glassmorphism) ve tam duyarlı (responsive) modern tasarım.

## 🚀 Yerel Geliştirme Kurulumu

Projeyi kendi bilgisayarınızda çalıştırmak için:

```bash
# 1. Repoyu klonlayın
git clone https://github.com/Anogar-Dxilak/Sitem.git

# 2. Proje dizinine gidin
cd Sitem

# 3. Bağımlılıkları yükleyin
npm install

# 4. Geliştirici sunucusunu başlatın
npm run dev
```

Tarayıcınızda `http://localhost:5173` adresini açarak siteyi görüntüleyebilirsiniz.
